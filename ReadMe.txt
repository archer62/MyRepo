WPF приложение, использующее API сервиса Eventful
Предназначено для поиска различных событий (выставок, лекций, концертов и т.д.)
Поле What предназначено для ключевых слов, по которым идет поиск (необязательно)
Поле Where предназначено для места (необязательно)
Поле From - начало периода (необязательно)
Поле To - конец периода (необязательно)
В листбокс выводится список найденных ивентов
В самом низу окна выводится информация о конкретном ивенте при нажатии на соответствующий ивент
